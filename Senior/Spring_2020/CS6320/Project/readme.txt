## NOTE

Please have the train.xml and text.xml in the same folder as project.py

Packages sklearn, pytorch and python3 are required to run this project.

All model code can be found within project.py. To run code, please use run.py.

## EXECUTION

Please type python run.py -h to see descriptions of these variables.

python  run.py [-h] [--N [N]] [--L [L]] [--max [max]] [--B [B]] [--S [S]]
               [--HD [D]] [--lr [lr]] [--disable_display]

Example:

python run.py -N 10 -L 2 --max 30
