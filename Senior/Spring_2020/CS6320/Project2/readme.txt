# SETUP

Please have the test and train sets "modified_test.txt" and "modified_train.txt" in the same directory as project.py
Python 3 was used for this project with the following libraries:
       - nltk
       - sklearn
       - scipy
       - pandas
       - numpy
Please have these installed
There will be prompts for you to download nltk packages, please do so, otherwise the lemmatizer and pos_tagger wont work.

# Execution
If you follow the steps above, all you need to do is run:

python program.py
